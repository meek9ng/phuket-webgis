01About

An island,
observed at 250 m resolution.

A 543 km² island province on Thailand's southwest Andaman coast, made up of three districts and home to one of Southeast Asia's most active tourism economies.

Phuket sits in the Andaman Sea, connected to mainland Thailand by the Sarasin Bridge. Its three districts — Mueang Phuket, Kathu, and Thalang — host beaches, dense urban centres, mountainous interior forest, and rapidly expanding peri-urban zones.

Since 2010, Phuket has experienced explosive urbanization driven by tourism, second-home development, and infrastructure investment. This has transformed the land cover, increased urban-heat-island pressure, and reshaped accessibility to public services.

This WebGIS provides a transparent, data-driven view of where change is happening, how fast, and how it affects livability — across every 250 m × 250 m cell of the island, every year from 2018 through 2026.

04Workflow

Nine steps,
raw pixels
to live map.

From raw Sentinel-2 scenes to a polished interactive WebGIS — each step feeds the next, with all intermediate outputs preserved for reproducibility.

Acquire raw data
Download Sentinel-2 L2A scenes (one cloud-free composite per year), the OpenStreetMap extract for southern Thailand, and the Phuket administrative polygon.
Copernicus Geofabrik GADM
Pre-process rasters
Apply cloud masks, build a median composite per year, classify pixels into 4 classes (built / vegetation / water / bare) using a supervised model, then reproject to UTM Zone 47 N (EPSG:32647).
rasterio scikit-learn numpy
Build the analysis grid
Generate a 250 m × 250 m fishnet over the study area (9 644 cells), then run zonal statistics from each yearly raster to populate per-cell land-cover fractions plus NDVI and UHI proxy.
geopandas shapely zonal stats
Enrich with OpenStreetMap
Spatial-join 244 016 OSM POIs into 5 categories per cell, compute distances to nearest medical / education / main road, derive POI density, Gini-Simpson diversity, and a composite livability score (0–100).
osmnx spatial join livability
Export web-ready files
Re-project to WGS84, write GeoJSON for grid / POIs / roads, render PNG image overlays, and emit summary CSVs and a single JSON of headline KPIs.
GeoJSON PNG CSV
Build the FastAPI backend
Expose /api/stats, /api/grid, /api/pois, /api/roads and 5 chart endpoints; mount /data for static files; serve the SPA from the root.
FastAPI CORS StaticFiles
Build the frontend
Wire Leaflet with a transparent canvas overlay (strips background pixels), a 250 m grid choropleth, POI clusters, year animation, hover tooltip, floating side panels, and SVG charts.
Leaflet SVG vanilla JS
Run & iterate locally
Serve via uvicorn main:app --reload --port 8080, debug with browser DevTools, profile slow endpoints, and refine UX based on usage.
uvicorn DevTools
Deploy
Containerise with Docker, host on Render / Fly.io / DigitalOcean, or pre-render data and host a fully static mirror on GitHub Pages + Netlify Functions.
Docker Render GitHub Pages
